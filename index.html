In this regard ESP32 based wifi connectivity, Arduino in combination with Bluetooth modules, Raspberry pi based semi- autonomous boats were developed.Various types of propulsion units such as azimuthal thrusters, water jet, twin screw, single screw designs were implemented using 3d printed parts which were designed using CAD tools.

River Sediment Explorer (RSE)

A project where I am conducting the comparison of suitability of either a SWATH or a Catamaran like hull for the base of a bathymetric ASV which would utilise SLAM based navigation algorithms for creating predictive maps for siltation patterns in Indian rivers. The project aims to facilitate increase of effective navigational depth of Indian rivers. Hydrostatics of the two hull forms have been estimated, their models have been designed and the navigational SLAM algorithm has been tested and implemented on a small scale.

Analysis and Validation of Fresnel-Lens based Solar Heat Accumulator for Auxiliary System of Passive Solar Distillation Device

A project that aimed to validate through numerical analyses the complete design of an auxiliary system which when operates in conjunction with a previously designed passive desalination device, provides not just a drastic enhancement in freshwater output but also complete self-sufficiency, which would be particularly beneficial for coastal economies. Empirical formulas to scale the model to any desired size based on freshwater requirement.
